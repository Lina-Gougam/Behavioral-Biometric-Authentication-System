Project Overview

Project: P003 — Behavioral Biometric Authentication System
Component: Touchscreen Gesture Authentication
Course: Computer & Network Security

This system authenticates users by analysing how they interact with a touchscreen — not what they know or have. Even if credentials are stolen, an attacker cannot replicate the unique behavioral signature of the genuine user.

System Pipeline

1. Raw Touch Events
x, y, pressure, size, timestamp collected at ~100 Hz
2. Feature Extraction
19 features per stroke: spatial, temporal, kinematic, pressure, size
3. User Profiling
One-class model trained on genuine user data (70/30 split)
4. Authentication Decision
Anomaly score vs EER threshold → Accept or Reject
